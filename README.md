# Sistema de Televisão em Python

Projeto desenvolvido em Python utilizando Programação Orientada a Objetos (POO).

## Funcionalidades

* Troca de canais
* Controle de volume
* Lista de canais disponíveis
* Menu interativo no terminal
* Validação de canais existentes
* Controle remoto integrado à TV

## Tecnologias utilizadas

* Python
* POO (Classes e Objetos)

## Estrutura do projeto

* `main.py` → interface principal
* `funcionalidades.py` → classes da TV e do controle remoto

## Conceitos aplicados

* Encapsulamento
* Associação entre objetos
* Métodos
* Condicionais
* Loops
* Validação de dados
